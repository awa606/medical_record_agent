# v1.3.3 医生端入口降噪验收记录

## 目标

回应老师反馈“模型选择医生不一定懂、医生端默认信息过多”，把右上角默认入口从工程化的“输入方式”收敛为医生更容易理解的“开始生成”。

## 本次调整

- 右上角主入口显示为 `开始生成`。
- 展开菜单保留三类动作：`上传音频`、`粘贴文本`、`Mock 演示`。
- `Mock 演示` 自动切换到 `mock` ASR，并打开音频生成抽屉，用于现场评审保底。
- 不修改后端 API、ASR/SSE、病历生成、医生审核和导出流程。
