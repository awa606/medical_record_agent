## 截图建议

- 优先使用 `http://127.0.0.1:8000` 本地页面截图。
- doctor.html 主界面截图建议使用“医生模式 + 截图模式”，隐藏不必要的调试按钮。
- Agent Trace 截图需要切换到“调试模式”。
- debug.html 截图只用于课程调试与评分证据展示，不作为真实医生端界面。
- 截图中不要出现真实 API Key、真实患者姓名、真实手机号、真实身份证号或真实医院系统信息。
- 如果使用 Online LLM，截图只展示 provider、model、fallback 状态，不展示环境变量值。
- 如果 `fever_01.wav` 现场 ASR 卡顿，可以使用已有运行日志截图作为备用证据。
