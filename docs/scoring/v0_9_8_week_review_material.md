## 9. 现场讲稿简版

本项目是一个面向中文医患问诊的 Medical Record Agent。它不是自动诊断系统，而是帮助医生把问诊音频或文本整理成可审核的电子病历草稿。当前系统已经打通了文本/音频输入、实时转写、角色校正、病历生成、质量评估、候选诊断提示、医生审核和导出。

本周重点不是单纯做页面，而是把工程闭环做完整：ASR 和说话人处理提供输入，病历质量规则保证字段可信，候选诊断和治疗建议都保留医生确认边界，Docker 和 GitHub Release 保证可以复现和追踪。当前还不是正式 `v1.0`，下一步会冻结最终文档和验收结果。边缘端部署已经有配置建议，但需要医院真实设备和合规条件明确后再做实机部署。
