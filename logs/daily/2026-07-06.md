## 追加记录：v0.3 医生/患者角色校正

### 今日计划

- 实现医生/患者角色逐段校正。
- 支持医生编辑转写文本并保存到 ASR session 结果。
- 保持 `/api/audio/{audio_id}/generate-record` 兼容校正后的 transcript。
- 明确当前模型路线：v0.3 不换模型，v0.5 再做本地模型和边缘端评测。

### 今日完成

- 新增 ASR segment 校正字段：`needs_review`、`reviewed_by_doctor`、`original_text`。
- 新增 `PATCH /api/asr/sessions/{session_id}/result`，保存角色和文本校正。
- 后端保存校正后同步更新 ASR session result 和旧音频 transcript。
- 医生端中间转写栏支持角色下拉、文本编辑、待确认状态和“保存角色校正”。
- 生成病历前会阻止未确认角色直接进入病历生成。
- 新增 `docs/asr_role_correction.md`、`docs/asr_model_route.md`。

### 问题与阻碍（Blocker）

- 当前 v0.3 仍是结果层校正，不是底层 diarization；多人物自动分离需要 v0.5 模型评测后决定。

### 解决方案 / 需求支持

- 保留人工校正入口作为医生最终确认边界。
- 将方言、多国语言、多人物转写放入 `docs/asr_model_route.md` 和 `docs/local_model_edge_benchmark.md` 评测路线。

### 明日计划
